# Crawling_PyQt

Pyqt5를 이용한 크롤링 검색 프로그램.

##### 목표
  * 조회조건에 해당하는 데이터 추출.

##### Environment
  * Python 3
  * Node.js - 개인 Synology Nas Server 
  * Maria DB
